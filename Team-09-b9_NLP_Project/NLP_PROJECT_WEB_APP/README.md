# 🎯 Automatic Meeting Minutes Generator

An AI-powered web application that converts meeting audio into structured and actionable insights.

---

## 🚀 Features

- 🎤 Upload meeting audio files (WAV, MP3, FLAC, OGG, M4A)
- 📝 Automatic transcription using OpenAI Whisper
- ✨ Smart summary generation using Transformer models
- 📌 Extraction of key highlights and insights
- ✅ Automatic action item detection (Task, Deadline, Priority)
- 📊 Word count and duration estimation
- 📥 Export results as TXT, CSV, and PDF

---

## 🛠️ Tech Stack

- **Frontend & App Framework:** Streamlit  
- **Speech Recognition:** OpenAI Whisper  
- **NLP Models:** Hugging Face Transformers (BART / DistilBART)  
- **Text Processing:** spaCy, TextBlob  
- **PDF Generation:** ReportLab
